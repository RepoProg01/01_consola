Comandos para actualizar e informacion

Comando Descripcion
sudo apt-get update Actualiza el software de Ubuntu
sudo apt-get upgrade Actualiza el software de Ubuntu
echo $0 Muestra que shell estas usando
whoami Muestra tu nombre de usuario
clear or (ctrl+L) Limpiar consola
file [archivo] Muestra informacion del archivo
lsb_release -a Muestra la version de Ubuntu
histori Muestra los comandos que se ha ejecutado
pwd Imprime la ruta en la que te encuentras
uname +
  • -a
  • -s (or) --kernel-name
  • -r (or) --kernel-release
  • -p (or) --processor
  • -m (or) --machine
  • -i (or) --hadware-platform
  • -n (or) --nodename
  • -o (or) --operateing-system

-a Muestra toda la informacion.

-s (o) --kernel-name Muestra el nombre del kernel(nucleo).

-r (o) --kernel-release Muestra version del kernel.

-p (o) --processor Muestra informacion del procesador.

-m (o) --machine Muestra arquitectura del procesador.

-i (o) --hardware-platform Muestra plataforma de hardware.

-n (o) --nodename Muestra nombre de red (maquina).

-o (o) --operating-system Muestra nombre de sistema operativo.

Comandos para cambiar o manipular carpetas/archivos

Comando Descripcion
cd [carpeta] Cambio de directorio, cd sin parametros lleva a Home
cd .. Salir de directorio
cd ./folder1/folder2 Ir a un directorio o archivo especifico
ls +
  • -l
  • -a
  • -h
  • -S
  • -r
  • -t

-l Muestra archivos en lista.

-a Muestra archivos ocultos.

-h Muestra archivos en kb.

-S Muestra informacion del procesador.

-r Muestra archivos en forma recursiva.

-t Muestra archivos por fecha mas reciente.
code . Abre la carpeta donde te encuentras en VSCode
tree +
  • -L 1,2 3 etc

-L Nivel.

1, 2, 3 etc Numero de nivel.
mkdir [carpeta] Crea una carpeta con el nombre que le des
touch [carpeta] Crea un archivo con el nombre y la extensión que le des
mv [archivo/carpeta] [destino] Renombra o Mueve una carpeta o archivo hacia un nuevo destino
cp [archivo] [destino] Copia un archivo hacia un nuevo destino
rm +
  • -i
  • -r
  • -f

-i Muestra confirmacion antes de borrar archivo.

-r Para borrar folder vacio.

-f Para forzar a borrar.

Comandos para archivos de texto plano

Comando Descripcion
head [archivo] Muestra las primeras 10 líneas de un archivo de texto
head -n 20 [archivo] Muestra las primeras 20 líneas
tail [archivo] Muestra las últimas 10 líneas
tail -n 20 [archivo] Muestra las últimas 20 líneas
less [archivo] Explora el contenido del archivo (y para salirte de este modo usas la Q)
